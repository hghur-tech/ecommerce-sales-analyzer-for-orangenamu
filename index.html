사전 안내
이 절차는 My Page 설정 완료 후 사용합니다. 아래 설정 방법을 먼저 수행하세요.
🔧 My Page 설정 방법 (선행)
  1. 로그인 화면 우측 상단 프로필 클릭 → 개인기능설정 진입
  2. mypage설정 클릭
  3. 왼쪽 mypage 기본그룹에서 빠른 접속 메뉴 선택
    체크 경로: 재고1 → 출력물 → 재고현황 → 재고변동표 선택 후 ← 버튼으로 기본그룹에 이동
  4. 재고변동표 화면에서 우측 +로 새 탭 생성 → 탭 이름을 당일재고실사용으로 변경
  5. 창고지정을 지하창고(오렌지나무)로 설정 → 저장/검색
  6. 화면 하단 검색 | 금일 | 전일 ... 영역에서 우측 아래 꺾쇠 아이콘으로 확장 → 설정 → 간편검색명 금일 우측 조회기본값 체크 → 저장

🚀 실행 순서

  1. 🔗 접속 — https://login.ecount.com/Login/
  2. 👤 로그인 — 회사코드 / 아이디 / 비밀번호 입력
  3. 🧭 이동 — My Page 하위 메뉴의 재고변동표 클릭
  4. 🏷️ 조건 확인 — 창고: 지하창고(오렌지나무), 기준일자: 오늘 ~ 오늘 상태 확인 후 검색
  5. 📤 다운로드 — 조회 결과 하단의 Excel 버튼 클릭 → 파일 저장
    표준 파일명: 재고실사표_YYYYMMDD_HHMM.xls
  6. 🧽 데이터 정리 — 엑셀을 열어 E열(출고수량)이 없는 행을 하단으로 모아 행 삭제
  7. ✅ 실사 대상 확정 — 남아있는 행의 SKU가 당일 재고실사 대상
  8. 📍 현장 실사 — 단품 위치/카톤 박스 위치 확인 후 실사 진행
⚠️ 주의
행 삭제 전 파일을 별도 사본(예: 재고실사표_RAW_YYYYMMDD_HHMM.xls)으로 보관하세요.

🗂️ 파일명 규칙

재고실사표_YYYYMMDD_HHMM.xls (예: 재고실사표_20250825_1030.xls)
